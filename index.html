alter table config enable row level security;

  for all to anon using (true) with check (true);

Regole generali

  • Prima riga = intestazioni (nomi colonne)
  • Separatore: virgola , o punto e virgola ; (rilevato automaticamente)
  • Codifica: UTF-8 (anche con BOM da Excel)
  • Se un campo contiene virgole, racchiudilo tra "virgolette"
  • Prodotti con EAN già presenti vengono aggiornati (upsert)
